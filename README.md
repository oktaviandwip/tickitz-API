## 🛠️ Installation Steps

<b>1. Clone the repository</b>

```bash
git clone https://github.com/oktaviandwip/tickitz-backend
```

<b>2. Install dependencies</b>

```bash
npm install
# or
yarn install
```

<b>3. Run the app</b>

```bash
npm start
```

🌟 You are all set!
